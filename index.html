LoopOS UI Foundations Transition

This page serves as a visual reference for transitioning our design tokens and styles from UI 1.0 to UI 2.0. The goal is to help you understand exactly how each token is migrating so that we can align design and code seamlessly and reduce conflicts during the rollout of the new UI.
